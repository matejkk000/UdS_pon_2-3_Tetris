NÁZOV PROJEKTU:
Tetris

CIEĽ PROJEKTU:
Cieľom projektu je vytvoriť funkčnú a prehľadnú verziu klasickej hry Tetris
pomocou jazyka Java a knižnice Swing. Projekt demonštruje prácu s grafickým
rozhraním, objektovým návrhom, spracovaním vstupov používateľa a základnou
hernou logikou (kolízie, odstraňovanie riadkov, skóre, pád blokov).

SPUSTENIE PROJEKTU:
Spusťte projekt z triedy Main.java, ktorá vytvorí hlavné okno a inicializuje
hernú plochu. Hra sa automaticky spustí po potvrdení dialógového okna.

POUŽÍVATEĽSKÁ PRÍRUčKA:
Pomocou šípok ovládate padacie bloky (doľava, doprava, otáčanie). Medzerník
okamžite zhodí blok úplne dole. Cieľom je vytvárať plné riadky, získavať body a 
zabrániť tomu, aby sa bloky dostali až na vrch hernej plochy.
